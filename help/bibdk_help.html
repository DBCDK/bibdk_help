This module creates a 'userhelp' content type with two viewmodes, a panel page, and a block with a 'user-help-help' menu, where links open in a popup window.

The 'user-help-help' menu is by default populated with a link to a page that lists the 'userhelp' nodes

The 'user-help-help' menu can have link to additional userhelp nodes (like: 'contact' and 'about').

The help panel page also has a search block, and the search is delimited to 'userhelp' nodes.

After installing, copy the files node--userhelp.tpl.php and page--userhelp.tpl.php to the active theme's '/templates' folder
We use $content['help_url'] instead of $node_url, in order to make the links in the bibdk_help_list viewmode point to a '/help/' path instead of a '/node/' path.
